
# macOS Setup

## Starship install

```bash
curl -sS https://starship.rs/install.sh | sh
```

## Homebrew Install

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

## go-sockaddr
```bash
go install github.com/hashicorp/go-sockaddr/cmd/sockaddr@latest
```
